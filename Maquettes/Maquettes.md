## 🌍 À Propos

**Taghazout v2** est une plateforme web **full-stack** dédiée à la digitalisation des services touristiques de **Taghazout** — village de surf emblématique sur la côte atlantique marocaine.

Elle centralise en un seul endroit tout ce dont un voyageur a besoin :

```
🏨 Hôtels  ·  🏄 Surf  ·  🍽️ Restaurants  ·  🎯 Activités  ·  💳 Paiements
```

L'objectif : offrir une expérience **rapide**, **sécurisée** et **intuitive** — aussi agréable que les vagues de Taghazout elle-même.

## ⚡ Installation

### Prérequis

```bash
✅ PHP 8.2+
✅ MariaDB 10.4+
✅ Composer
✅ Compte Stripe (gratuit)
```

### 1️⃣ Cloner le Repository

```bash
git clone https://github.com/zakariyaaaaaa/Taghazout_v2.git
cd Taghazout_v2
```

### 2️⃣ Installer les Dépendances

```bash
composer install
```

### 3️⃣ Configurer l'Environnement

```bash
cp .env.example .env
```

Puis éditer `.env` :

```env
DB_HOST=localhost
DB_NAME=taghazout
DB_USER=root
DB_PASS=

STRIPE_SECRET_KEY=sk_test_VOTRE_CLE_ICI
STRIPE_PUBLIC_KEY=pk_test_VOTRE_CLE_ICI
```

### 4️⃣ Importer la Base de Données

```bash
mysql -u root -p taghazout < database/taghazout.sql
```

### 5️⃣ Lancer le Serveur

```bash
php -S localhost:3000
```

```
🌐  Ouvrir : http://localhost:3000
```
